This is an Android WebApp to display the mobile website created by a python Script [1] which 
is intended to be used  with the XML output of Pentabarf [2] or frab [3].

The version in this repository will be used for the FrOSCon [4] so you need to modify some 
FrOSCon related links.

[1] https://github.com/derpeter/Confernce-Mobile-Schedule
[2] http://www.pentabarf.org
[3] https://github.com/oneiros/frab
[4] http://froscon.org
